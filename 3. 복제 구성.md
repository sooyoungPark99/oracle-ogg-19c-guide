## 2. GGSCI 초기 설정 (Manager)

```bash
cd /app/oracle/goldengate/19c
./ggsci
```

```
GGSCI> CREATE SUBDIRS

GGSCI> EDIT PARAMS MGR
```

MGR 파라미터 파일 내용 (`:wq`로 저장):

```
PORT 7809
DYNAMICPORTLIST 7810-7820
AUTORESTART EXTRACT *,  RETRIES 5, WAITMINUTES 1
AUTORESTART REPLICAT *, RETRIES 5, WAITMINUTES 1
PURGEOLDEXTRACTS ./dirdat/*, USECHECKPOINTS, MINKEEPDAYS 3
```

```
GGSCI> START MGR
GGSCI> INFO MGR
-- Manager is running (IP port TCP:asmtest.7809 ...) 이면 정상
```

## 3. Extract 설정

```
GGSCI> EDIT PARAMS EXT1
```

```
EXTRACT EXT1
USERID oggadmin@orcl, PASSWORD oracle
EXTTRAIL ./dirdat/lt
TABLE scott.*;
```

```
GGSCI> DBLOGIN USERID oggadmin@orcl PASSWORD oracle
-- Successfully logged into database. 이면 정상 (WARNING OGG-25108은 무시)

GGSCI> ADD SCHEMATRANDATA scott
-- INFO OGG-01788 이면 정상

GGSCI> ADD EXTRACT EXT1, TRANLOG, BEGIN NOW
GGSCI> ADD EXTTRAIL ./dirdat/lt, EXTRACT EXT1, MEGABYTES 500
GGSCI> START EXTRACT EXT1
GGSCI> INFO EXTRACT EXT1
-- Status: RUNNING 이면 정상
```

## 5. Replicat 설정

### 체크포인트 테이블 생성

```
GGSCI> DBLOGIN USERID oggadmin@clonedb PASSWORD oracle

GGSCI> ADD CHECKPOINTTABLE oggadmin.ggs_checkpoint
-- Successfully created checkpoint table 이면 정상
```

### REP1 파라미터 파일 생성

> **ASSUMETARGETDEFS, CHECKPOINTTABLE**은 파라미터 파일에 넣지 않는다.

```
GGSCI> EDIT PARAMS REP1
```

```
REPLICAT REP1
USERID oggadmin@clonedb, PASSWORD oracle
MAP scott.*, TARGET scott.*;
```

### Replicat 등록 및 시작

```
GGSCI> ADD REPLICAT REP1, EXTTRAIL ./dirdat/rt, BEGIN NOW, CHECKPOINTTABLE oggadmin.ggs_checkpoint
GGSCI> START REPLICAT REP1
GGSCI> INFO ALL
```

정상 결과:

```

EXTRACT     RUNNING     PMP1        00:00:00      00:00:04
```
