# 0. 사전 준비

orcl DB는 Oracle 19c 설치 시 이미 생성되어 있다.
OGG 복제 대상인 clonedb를 DBCA로 별도 생성한다.

---

## 1. orcl DB 기동 확인

```bash
ps -ef | grep pmon
# ora_pmon_orcl 이 떠 있어야 한다
```

```bash
export ORACLE_SID=orcl
sqlplus / as sysdba
```

```sql
select instance_name, status from v$instance;
-- OPEN 이어야 한다

exit;
```

---

## 2. DBCA로 clonedb 생성

> 메모리가 부족하면 orcl DB를 종료 후 생성하고, 완료 후 다시 기동한다.

```bash
dbca
```

DBCA GUI 진행 순서:

1. **Create a database** 선택
2. **Advanced configuration** 선택
3. **Oracle Single Instance database** 선택
4. **General Purpose** 템플릿 선택
5. 데이터베이스 정보 입력
   - Global database name: `clonedb`
   - SID: `clonedb`
   - **컨테이너 DB(CDB) 체크 해제**
6. 스토리지 설정
   - Storage type: **File System** 선택
   - Database files location: `/u01/app/oracle/oradata`
7. Fast Recovery Area
   - Recovery files location: `/u01/app/oracle/fast_recovery_area`
   - Recovery area size: `10GB`
8. Network Configuration
   - 기존 LISTENER(1521) 가 목록에 표시되면 그대로 체크 후 Next
9. 샘플 스키마 설치 체크 (Sample schemas 탭 → Add sample schemas 체크)
10. 메모리 설정 (Configuration Options → Memory 탭)
    - **Use Automatic Shared Memory Management** 선택
    - SGA size: `700MB`
    - PGA size: `300MB`

    > **[DBT-11214] Automatic Memory Management is not a feasible option** 오류 발생 시
    > Use Automatic Memory Management 대신 Use Automatic Shared Memory Management 를 선택한다.

11. 비밀번호: `oracle` (모든 계정 동일)
12. **Finish** 후 생성 완료 확인

## 3. clonedb 생성 확인

```bash
ps -ef | grep pmon
# ora_pmon_orcl 과 ora_pmon_clonedb 둘 다 있어야 한다
```

```bash
export ORACLE_SID=clonedb
sqlplus / as sysdba
```

```sql
select instance_name, status from v$instance;
-- clonedb OPEN

exit;
```

## 4. tnsnames.ora 확인

```bash
vi $ORACLE_HOME/network/admin/tnsnames.ora
```

아래 내용이 있는지 확인하고 없으면 추가한다.
HOST는 서버 실제 IP로 입력한다. orcl, clonedb 모두 동일한 서버이므로 IP는 같다.

```
ORCL =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = 172.31.0.250)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = orcl)
    )
  )

CLONEDB =
  (DESCRIPTION =
    (ADDRESS = (PROTOCOL = TCP)(HOST = 172.31.0.250)(PORT = 1521))
    (CONNECT_DATA =
      (SERVER = DEDICATED)
      (SERVICE_NAME = clonedb)
    )
  )
```

```bash
tnsping orcl
tnsping clonedb
# 양쪽 모두 OK 나와야 한다
```

## 5. SID 전환 alias 추가 (선택)

```bash
vi ~/.bash_profile
```

```bash
alias orcl='export ORACLE_SID=orcl'
alias clonedb='export ORACLE_SID=clonedb'
```

```bash
source ~/.bash_profile
```
