# 1. OGG 설치 및 환경설정

---

## 1. OGG 다운로드

Oracle eDelivery 사이트에서 다운로드한다.

```
https://edelivery.oracle.com
```

- Product: `Oracle GoldenGate`
- Release: `19.1.0.0.x`
- Platform: `Linux x86-64`
- 다운로드 파일명: `V983658-01.zip` (약 530MB)

> Oracle 계정(무료)이 필요하다.

---

## 2. zip 파일을 서버로 전송

```bash
# Windows → Linux 서버로 전송 (압축 해제하지 말 것)
scp V983658-01.zip oracle@192.168.13.181:/home/oracle/
```

---

## 3. 서버에서 압축 해제

```bash
su - oracle
cd /home/oracle
unzip V983658-01.zip
```

압축 해제 후 구조:

```
/home/oracle/
├── fbo_ggs_Linux_x64_shiphome/
│   └── Disk1/
│       └── stage/Components/
├── OGG_WinUnix_Rel_Notes_19.1.0.0.4.pdf
└── OGG-19.1.0.0-README.txt
```

> ⚠️ 이 버전은 OUI 방식이다. 내부에 별도 zip 파일은 없다.
