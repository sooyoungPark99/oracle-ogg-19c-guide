# 1. OGG 설치 및 환경설정

---

## 1. OGG 다운로드

Oracle eDelivery 사이트에서 다운로드한다.

```
https://edelivery.oracle.com
```

- Product: `Oracle GoldenGate`
- Release: `19.1.0.0.x`
- Platform: `Linux x86-64`
- 다운로드 파일명: `V983658-01.zip` (약 530MB)

> Oracle 계정(무료)이 필요하다.

---

## 2. zip 파일을 서버로 전송 (MobaXterm)

MobaXterm 왼쪽 패널의 파일 탐색기를 이용해 업로드한다.

1. MobaXterm 접속 후 왼쪽 **파일 탐색기 패널** 확인
2. 경로를 `/home/oracle/` 로 설정
3. Windows 탐색기에서 `V983658-01.zip` 파일 선택 후 업로드

<img width="1110" height="338" alt="image" src="https://github.com/user-attachments/assets/c4f43557-0753-4ec7-96b0-651c7c65e10c" />

---

## 3. 서버에서 압축 해제

```bash
su - oracle
cd /home/oracle
unzip V983658-01.zip
```

압축 해제 후 구조:

```
/home/oracle/
├── fbo_ggs_Linux_x64_shiphome/
│   └── Disk1/
│       └── stage/Components/
├── OGG_WinUnix_Rel_Notes_19.1.0.0.4.pdf
└── OGG-19.1.0.0-README.txt
```

## 4. oraInst.loc 파일 생성

```bash
cat > /home/oracle/oraInst.loc << EOF
inventory_loc=/u01/app/oraInventory
inst_group=oinstall
EOF

cat /home/oracle/oraInst.loc

```

## 5. OGG 설치 홈 디렉토리 생성

```bash
mkdir -p /u01/app/oracle/goldengate/19c
```

---

## 6. runInstaller 실행 (Silent 모드)

```bash
cd /home/oracle/fbo_ggs_Linux_x64_shiphome/Disk1

./runInstaller -silent \
  INSTALL_OPTION=ORA19c \
  SOFTWARE_LOCATION=/u01/app/oracle/goldengate/19c \
  -invPtrLoc /home/oracle/oraInst.loc \
  START_MANAGER=false
```

설치 완료 메시지:

```
Successfully Setup Software.
Oracle GoldenGate Core의 설치을(를) 성공했습니다.
```

## 7. 환경변수 설정

```bash
vi ~/.bash_profile
```

아래 내용을 추가한다.

```bash
export OGG_HOME=/u01/app/oracle/goldengate/19c
export LD_LIBRARY_PATH=$ORACLE_HOME/lib:$OGG_HOME:$LD_LIBRARY_PATH
export PATH=$OGG_HOME:$PATH
```

```bash
source ~/.bash_profile

# 확인
which ggsci
# 결과: /u01/app/oracle/goldengate/19c/ggsci
```
